Palatte

The palatte refers to the selction of placeable objects in the sidebar. The object "oPalatte" controls the logic and drawing of the palatte.

How the palatte object data

In the draw event of oPalatte, a gigantic switch statement holds the data for all the palatte elements. Each iteration of the for loop does the button for one of these elements. If you add, remove, or swap elements be sure that the number preceding each case statement is unique.

How the palatte draws and does logic

For each element, the palatte draws a button (with the appropriate sprite, etc) and checks if it was clicked this step. If it was clicked, it sets the selected_object and other variables in the object "oEdit."

Adding new palatte object checklist

Here is a checklist of things to do when adding a new palatte object.
Create normal sprite.
Create default skin sprite.
Create object. Be sure its mask is the default sprite.
Add to script objectIsEdit
Add to script editObjectSerialize
Add to script createFromSerialization
Add to script loadSkin
Add to object oPalatte
